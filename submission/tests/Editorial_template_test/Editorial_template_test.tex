\section{Editorial}\label{editorial}

\textbf{Führen Sie in die Ausgabe ein, indem Sie:}

\begin{itemize}
\item
  das Schwerpunktthema (z.B. Methode, Ansatz, Gegenstand) der Ausgabe
  kurz vorstellen,
\item
  auf die Relevanz des Themas für textbasierte (digitale)
  Geisteswissenschaften eingehen,
\item
  das Schwerpunktthema in seinen Froschungskontexten (Traditionen,
  aktuelle Entwicklungen, methodische Bedeutung) darstellen
\item
  Gehen Sie bei der Veröffentlichung von Lehrkonzepten bitte auch darauf
  ein, warum sich das Thema (besonders) für die Lehre geeignet ist (z.B.
  niederschwellig, kollaborativ, praxisorientiert)
\end{itemize}

\textbf{Gehen Sie auf die Entstehung der Ausgabe und ihre Verortung in
der Zeitschrift ein:}

\begin{itemize}
\item
  Was war Motivation/Anlass für das Thema der Ausgabe?
\item
  Warum passt das Thema gut zur Ausrichtung der Zeitschrift?
\item
  Gehen Sie darauf ein, welche Beitragstypen im Rahmen der Ausgabe
  veröffentlicht werden (z.B. nachnutzbare Lehrkonzepte,
  Methodenbeiträge)
\end{itemize}

\textbf{Gehen Sie kurz auf den Redaktions - und Reviewprozess ein:}

\begin{itemize}
\item
  Nennen Sie das Begutachtungsverfahrens (Open Peer Review, Zero-Blind)
  sowie
\item
  Beteiligte Akteur*innen (z.B. Fachredaktion, externe Gutachter*innen,
  Gastherausgeber*innen, \ldots)
\end{itemize}

\section{Beiträge der Ausgabe}\label{beitruxe4ge-der-ausgabe}

Sofern zutreffend, erklären Sie an dieser Stelle bitte kurz, dass neue
Beiträge im Rahmen des Rollings Issues fortlaufend veröffentlicht
werden.\\
\strut \\
\ul{Hinweis}: Werden die Beiträge der Ausgabe als Rolling Issue
veröffentlicht, so wird dieser Abschnitt mit jeder
Beitragsveröffentlichung um die jeweilige Beitragsbeschreibung
erweitert.

\textbf{\hfill\break
Beschreiben Sie die einzelnen Beiträge kurz, indem Sie je nach
Beitragstyp auf folgende Punkte eingehen:}

\begin{itemize}
\item
  Autor*innen, Titel
\item
  Thema/Fokus
\item
  Lehrsetting (Lehrveranstaltung, Workshop, online etc.), Methoden,
  Werkzeuge
\item
  Zielsetzungen
\item
  Mögliche Besonderheiten
\end{itemize}
