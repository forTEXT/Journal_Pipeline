\section{Inhalt}\label{inhalt}

\begin{enumerate}
\def\labelenumi{\arabic{enumi}.}
\tightlist
\item
  Definition
\item
  Anwendungsbeispiel
\item
  Geisteswissenschaftliche Tradition
\item
  Technische Grundlagen
\item
  Diskussion
\end{enumerate}

\section{1 Definition}\label{definition}

Definieren Sie die Methode in ca. 1-2 Absätzen (bis zu 150 Wörter).
Nennen und erklären Sie dabei zentrale Begriffe.

\section{2 Anwendungsbeispiel}\label{anwendungsbeispiel}

Unter diesem Punkt soll die Methode auf Grundlage eines
Anwendungsbeispiels in bis zu 300 Wörtern praxisnah vermittelt werden.

\section{3 Geisteswissenschaftliche
Tradition}\label{geisteswissenschaftliche-tradition}

Unter diesem Punkt soll der Bezug der Methode zur
(geisteswissenschaftlichen) Forschungstradition sowie ihr Mehrwert für
diese in bis zu 450 Wörtern dargestellt werden.

\begin{enumerate}
\def\labelenumi{\alph{enumi}.}
\item
  Legen Sie dar, inwiefern die Methode an (traditionelle)
  geisteswissenschaftliche Traditionen und Fragestellungen anknüpft.
\item
  Stellen Sie dar, inwiefern die Methode von (traditionellen)
  geisteswissenschaftlichen Traditionen und Fragestellungen abweicht.
\item
  Falls keine analoge Forschungstradition besteht, gehen Sie in diesem
  Teil darauf ein, wie sich die Methode auf geisteswissenschaftliche
  Fragestellungen anwenden lässt und warum sie sich dafür eignet.
\end{enumerate}

\section{4 Technische Grundlagen}\label{technische-grundlagen}

Unter diesem Punkt soll in bis zu 400 Wörtern vermittelt werden, wie
niederschwellig die Methode ist und wie aufwändig ihre
Anwendung/Umsetzung ist.

\begin{enumerate}
\def\labelenumi{\alph{enumi}.}
\item
  Nennen Sie technische Kenntnisse, die zur Anwendung der Methode nötig
  sind. Gehen Sie in diesem Zusammenhang auch auf die
  Niederschwelligkeit der Methode für technisch weniger geschulte
  Nutzer*innen ein.
\item
  Gehen Sie darauf ein, wie viele und welche Ressourcen die Methode in
  Anspruch nimmt. Dies können bspw. Rechenkapazitäten ebenso wie der
  zeitliche Aufwand zur Aneignung/Umsetzung der Methode sein.
\item
  Gehen Sie ggf. auf Besonderheiten der Implementierung der
  vorgestellten Methode ein.
\item
  Nennen Sie weiterführende Quellen oder Ressourcen.
\end{enumerate}

\section{5 Diskussion}\label{diskussion}

Gehen Sie in bis zu 450 Wörtern auf verschiedene
Ausprägungen/Implementierungen der vorgestellten Methode und/oder
ähnliche Methoden/Ansätze ein. Es sollen insbesondere Potenziale und
Herausforderungen reflektiert und dargestellt werden.

\begin{enumerate}
\def\labelenumi{\alph{enumi}.}
\item
  Gehen Sie auf Varianten oder Ausprägungen der Methode ein.
\item
  Gehen Sie ggf. vergleichend auf ähnliche Methoden ein.
\item
  Diskutieren Sie Potenziale, Stärken, Schwächen, Herausforderungen
  und/oder Besonderheiten der Methode.
\end{enumerate}
